    读完了《人类简史》 #

    最后讲现代人类拥有了前人无法想象的优渥生活条件，但我们真的更快乐吗？快乐的本质是什么？

    生物学派把快乐的本质理解为快感，而人体中分泌的快感是恒定的，所以我们无法得到更多快乐。

    佛学认为我们不应该追求快乐，因为追求快乐就会回避痛苦，但痛苦是无法回避的。越执着于快乐，越深陷痛苦。所以我们应该放弃主观感受，让快乐来、快乐去、痛哭来、痛苦去。

    这很有趣，冥想就是这套理论。但还是有点费解，放弃追求主观感受，活着的意义是什么？查了一下，佛教又强调也要放弃追求生命的意义。慢慢悟吧。

    下午和小狗一起去大松果攀岩，线还是那么难，但是爬到一半的时候突然出现一个男人持续地给我们指导，还指出了我们的问题是手拉攀岩而不是用脚发力。后面得知他是大松果的老板，他给我们讲了差不多一个小时！要点：

    1. 攀岩是像走路一样，先用脚移动重心而不是用上半身；所以在发力时上半身要尽量轻松
    2. 合理的线路中脚滑是因为重心不对，而不是点滑，如果能正确地利用重心，不会出现脚滑的问题
    3. 不锁臂的意思不是全部蹲下，把手打直，而是脚把身体推起来，要找这个感觉
    4. 很多时候我觉得憋可能是因为我的发力不对，把重心缩起来了，而不是真的憋
    5. 说了这么多如果记不住，在攀岩时注意这三点：
      1. 背打直——目的是保持上半身的稳定；同时手臂应该是像内夹紧，而不是手臂向外打开（与四柱的发力一致），这样才能让整个身体都参与进来而不是只用手拉
      2. 低头看脚——两个目的，一是把脚踩稳，重点发在脚上，二是低头看脚点时自然会把身体推远，把发力的位置让出来
      3. 抬头推胯送髋——发力模式 总得来说就是：先把上半身放松——踩脚——推胯伸手 这节课更大的意义在于让我知道了后续的训练方向。把掌握正确的发力技巧作为目的而不是刷线。

    昨晚读桑塔格的《反对诠释》，是她的一本批评文集。第一篇与书同名，讲她认为评价艺术的方式不应该是尝试解释艺术，而是分析艺术形式。老实说刚开始读这篇的时候越读越困，但是这个观点还挺吸引我的。她主张我们应减少对作品的抽象解读，转而关注艺术本身带来的直观、感性的体验。放弃追求意义，专注在艺术本身提供的真实感受。有魅力。
