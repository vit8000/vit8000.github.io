    运动记录 #

    瑜伽：瑜伽馆 6次；家里6次

    攀岩：10次（熊猫4 大松果1 丘山2 飞蛙2 西安斑马1）

    羽毛球：1次

    跑步：1次 大明宫

    告别羽毛球的一个月。明明上个月还说要一周打一次，说完后只打了一次。😆。

    这个月攀岩真正爱上了难度，2次去丘山都更享受爬难度而不是抱石。目前水平在10b上下，软的10c可以摸，但难的10b过不到。相对来说还是只能爬比较简单的线。夹角线似乎很适合身高高的人！过了一条夹角的10c，爬下来体感是10b。判断标准是：点都很好抓，没觉得哪里比较难，就是10a；点巨好抓，可以跳很多点，就是.9；有单步难点，不一定能过，就是10b；有很多难点，完全看不懂，就是10c。

    4月拉第一个引体计划失败了。没有每天拉。对日常居家训练坚持的不好。改成悬挂吧，每天悬挂两个30秒。

    跑步在西安跑了大明宫。真累呀，但体验不错，还想跑。

    书影音游 #

    游戏 #

    游戏 ｜苏丹的游戏 9.6h

    一个周目也没玩完！第一天玩时没玩明白，也没觉得好玩。又重开了几个档后终于走到应该是初中期的剧情了。但做好人压力太大，一点错事都不想干，导致又想重开。那个时候游戏还没有建多档的功能，就搁置了。4月下旬更新了建多档的功能，但还没开始玩新的。应该还是会再玩一下吧，还是有想继续看发展的冲动。希望下次玩时能更随性一点。至少打完一个完整的周目吧！狗苏丹的结尾我都玩到过好几次了。

    游戏 ｜动物园之星

    玩了很久很久。建了2、3个动物园了吧。建了好几次终于把一个巨大的河马栖地建好了。也大概搞明白了这个游戏该怎么玩。喜欢建不同的栖地把动物放进来。但不喜欢建给人的造景，也不喜欢构思员工路径。

    游戏 ｜租房达人

    我玩游戏真的是打工的命。又是一个打工、建造类游戏。扮演房东装修、租房，给房客们打工。游戏的第一个大目标是每个月流水8K，我有4个房子，但流水只有2K。看别人视频才发现别人一个房子就收2K租金，我收几百！下次打开游戏第一件事就是把房客都赶出去，重新装修一下重新租。

    PS 想把美国阿卡迪亚打完，需要波子帮我把3D的剧情过掉。

    书 推理小说月 #

    书 ｜《漫长的午后》

    日文版原名直译是《漫长的午后》，中文版翻译成了《恶女的告白》。很有想法的一步小说，也没想到是男作者写的。很喜欢开头的引入方式。

    书 ｜《秘密结晶》

    反乌托邦小说。比起反抗不成功，更可怕的事遗忘让人麻木到连任何反抗的欲望都没有。

    书 ｜《帷幕》

    阿加莎波洛系列最后一部。没有特别喜欢，但尊重。

    书 ｜《大师和玛格丽特》

    很妙。读了《痛疼部》后简单了解了一下东欧历史，对东欧文学作品也感兴趣起来。 很多隐喻读不懂，但还是很喜欢。更多的了解文化背景后再读一次。
