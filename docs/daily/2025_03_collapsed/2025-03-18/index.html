    2025-03-18

    早上去上了早课，今天感受到了上犬前面几个腰底很紧，拉开后再做就没那么紧了。

    和小猪模拟了一下周六去野攀给她打保护。有几个注意点：

    1. 落石锁脖子，用头盔保护，不松手
    2. 绳子需比室内更收紧（更安全）
    3. 前几米需提供保护，同时不能松制动手
    4. 退后时注意别摔了
    5. 给绳时不松制动手
    6. 多沟通，慢慢爬
    7. 绝不松制动手！！

    今天先吃饭，再回来看书睡觉，有点晕乎乎的。

    读完了《你要快乐，不必正常》。心疼作者，准备去看她的《橘子不是唯一的水果》

    下午工作也挺忙的，没什么空摸鱼。一忙起来就哪儿哪儿都不太舒服。

    晚上在家吃了波子做得土豆烧牛肉，很好吃，就是有点干了，有提升空间。

    整理了一下李静睿《死于昨日世界》中提到的想看的书，发现基本都是男性作者。不知道如果作者再写一本书评还会不会还是这个性别比例。
