    再探眉山 #

    周六再探眉山，这次去V2差不多可以爬完了。有一条很难的V2，过了！感受到眉山很多线是真卡身高，对我来说很简单。发力很舒服，没有哪条线感觉到憋。力量进步了，是因为最近吃了更多蛋白质吗？ 中午吃的油炸串串，晚上吃马旺子。

    爬15米的难度 #

    爬到一半都没到时，我就觉得我爬不动了。

    这时对着下面喊“我不行了，我要下来”，还会有回应：“再坚持一下，你确定要下来吗？”

    我沉默，又休息了一下，再接着爬了几米。

    “这回真不行了！”

    “多休息一会儿，别下来！”

    又爬了几个点，爬一步，掉一步，这个时候喊：“我真不行了！一点力气都没有了！“，下面不知道是隔得太远听不清，还是不想理我，已经完 全没有回应了。

    离地面越远，我给自己打气的声音就越大。最后一步边大喊“come on”，边发力牢牢抓住终点时，还是很爽的。

    那一刻我完全相信我自己的双手，也完全相信我的保护员。

    这就是攀岩的魅力吧。

    第一届厨艺大赛 #

    下午四人帮+2来家里做饭，好吃好玩。在家做饭实在是太开心了。又开展了运动大比武。 冠军居然是土豆烧排骨，亚军西红柿炖牛肉，季军青花椒鱼。

    独生子女的好处是长大后我们可以自己选择家人。和朋友们一起玩时，真的好希望大家可以都住在一起，随时串门。
