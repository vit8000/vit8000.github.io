    2025-02-17

    瑜伽 书

    瑜伽 #

    今天去教室上了早课，状态一般，身体有些疲惫，腿发不上力。

    早上五点钟就醒了，后面没怎么睡着。波子说我昨晚又出汗了，但我没醒，可能真有点感冒了。

    工作 #

    一想到今天要开早会就有点不开心。

    一上午基本都在写周报，写到12:30，坐在位子上动都没动，也没喝水，希望下周能快点儿弄完。

    食住行 #

    电脑本来马上组装完了，收的二手显卡特别黄，退了又重新买了一个，周五应该可以玩上了！

    天府大道4根车道变5根了，再也不是我的快乐老家车道了。

    晚上邀请小猪来家里吃国宴，波子买了豪华五花肉做红烧豆豆肉，好吃。小猪带了卤牛肉和酒来。

    吃完饭一起看了《某种物质》，还挺好看的。通过暴力、猎奇的方式，展现人类对关注和爱的畸形渴望。

    书 #

    昨天读《看见》特别喜欢讲卢安克的这一篇。

    我问：“你原来也有过那种着急的要改变的状态，怎么就变了，就不那样了？” 
“慢慢理解为什么是这个样子，理解了就觉得当然是这样了。” 
“你对现实完全没有愤怒？”
“没有。”
“你知道还会有一种危险是，当我们彻底地理解了现实的合理性，很多人就放弃了。”这是我的困惑。 
“那可能还是因为想到自己要改变，所以没办法了，碰到障碍了，就放弃了。我也改变不了，但也不    用改变，它还是会变。”
“那我们做什么呢？” 
“把自己的事情做好。”

    “理解了就没有愤怒了”与斯宾诺莎[[2025-02-10]]所说很像。对世界认知充分了，就理解了，理解了就知道为什么存在了。
