    2025-03-07&周末

    书 攀岩
      • 周五
      • 周末

    周五 #

    早上赖床了，没起来。腿和屁股坐昨天康复课按后好痛。把屏幕使用时间改为提醒坐姿的频率：每10分钟，10s。

    小猫太好笑了，本来白可一直吃右面那盆猫粮，黑多吃左面的，这两天不知道发什么神经，白可要先去吃黑多那盆，把黑多挡住，不让黑多吃。

    糟了，周五下午没心情工作。

    明天是妇女节！我的节日！一年最喜欢的节日（之一）。

    开始读《我要快乐，不必正常》是《橘子不是唯一水果》的作者的自传。《橘子》本来就是半自传小说。题记是感谢三位母亲，读了10%，期待后面。

    据说双点博物馆是3代里面最好玩的一代，有点心动。但双点大学很一般呀，玩了2、3个图后就不想玩了。

    《初步举证》 #

    和小狗、波子一起看了《初步举证》，酣畅淋漓的一场表演。实在是太精彩了，90分钟充满力量感的独角戏，如果在现场看应该会精彩得昏过去。世界上怎么会有这么伟大的演员。剧本也很优秀，我如此相信的法律原来会因为我是女人而背叛我。曾经的我站在对面，但现在为了千千万万个你我，我不再沉默。谢谢你。

    周末 #

    妇女节快乐！ #

    波子请我吃了咖喱饭，谢谢波子。好久没进城了，城里好多人，有点不适应。成都确实挺美的。

    读完了《人类简史》 #

    最后讲现代人类拥有了前人无法想象的优渥生活条件，但我们真的更快乐吗？快乐的本质是什么？

    生物学派把快乐的本质理解为快感，而人体中分泌的快感是恒定的，所以我们无法得到更多快乐。

    佛学认为我们不应该追求快乐，因为追求快乐就会回避痛苦，但痛苦是无法回避的。越执着于快乐，越深陷痛苦。所以我们应该放弃主观感受，让快乐来、快乐去、痛哭来、痛苦去。

    这很有趣，冥想就是这套理论。但还是有点费解，放弃追求主观感受，活着的意义是什么？查了一下，佛教又强调也要放弃追求生命的意义。慢慢悟吧。

    It’s GAME time #

    和波子一起玩完了《leap year》，两个人玩比自己玩更好玩。作为一个跳跃游戏，“别跳！别跳！”是最常说的一句话。

    开始玩《姐妹成行》，好好玩！是一玩就控制不住时间想一直玩的游戏。
