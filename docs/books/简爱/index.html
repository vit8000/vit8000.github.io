    正直、勇敢、有阅读量的小简爱

    书
    • Thoughts
    • 摘录

    作者: 夏洛蒂·勃朗特

    国家: 英国

    分类: 长篇小说

    主题: 教育，爱情，友情，宗教，成长

    读完日期: 2025-01-30

    rating: 7/7 值得反复阅读

    Thoughts #

    《简•爱》出版于一八四七年，作者为克服性别偏见带来的阻碍，出版时使用了柯勒•贝尔这个笔名。

    《简·爱》如果是男人写的，就是一本杰作，如果是女人写的就让人震惊或者厌恶。

    甚至也有人曾说《简爱》是勃朗特姐弟共同完成的，其中关于病榻的细节、服装的惊喜描写是姐姐完成的，而其余宏大、深奥、复杂的部分是弟弟完成的。

    结合当时的时代背景来看，这本书显得更伟大了。

    故事 #

    这本1874年小说的女主角比当代许多影视、书籍中塑造的女性角色更加独立。简爱出身贫困，父母早亡从小被姨母收留，姨母和她的几个孩子对她很差，她隐忍了下来。有机会时毫不犹豫地选择了离开。比起姨母和几个表兄妹的恶劣，作者对姨母家的仆人贝西有更多的描写。贝西在其他人欺负简爱时并不会站出来帮她说话，会和他人一起斥责简爱太过调皮，但在简爱受伤生病时细心温柔地照顾她。简爱讨厌她的姨母一家，但留在她心里的是贝西。这让我读到了简爱的坚强和善良。

    离开姨妈家后，简爱被送到洛伍德寄宿学校（是某种基督寄宿孤儿院学校）。她结识了改变她性格的好友海伦·彭斯。海伦·彭斯比她更成熟、独立且虔诚。她们的关系很像波伏瓦的《形影不离》中初期希尔维和安德蕾的关系。简爱和希尔维的性格都被学校认识的朋友改变了许多。

    在洛伍德毕业后，简爱留校教了几年书。她的导师离开了学校时，她也凭着几年的教学经验找到了一份新工作。在命运的指引下，她来到了桑菲尔德庄园，成为罗切斯特家的家庭教师，教授他的养女阿戴尔。 简爱中对罗切斯特的外表没说几句好话（对简爱自己的外表也没说）。 她慢慢地和罗切斯特相爱了（罗比她大20岁！是不是有点怪怪的）。她爱上了罗的幽默、智慧，罗欣赏她的智慧、独立和热情。二人阶级落差极大，但他们的关系又建立在精神上的平等和相互尊重上。不要忘了，这本书可是写在1874年！

    简爱和罗切斯特经历了一番你来我往的互相试探后，决定结婚。没想到的是，在婚礼现场，有人提出反对这段婚姻，因为罗其实是有老婆的！他的老婆就是被关在桑菲尔德庄园阁楼里的“疯女人”伯莎·梅森。 罗年少时在家人的安排下与西印度群岛的富豪家族梅森结婚，这场婚姻由家族利益促成。罗见到梅森很美丽很心动，并且她的家族有疯子基因，她家里人为了尽快把她送走也希望二人立刻结婚。在她发疯之前二人也十分相爱，度过一些愉快的时光。但回到桑菲尔德庄园没多久，她就彻底变成了一个疯子，基本没有理智，见谁咬谁。罗出于无奈把她锁在了阁楼里，并对外隐瞒了已婚的事实。

    （《阁楼上的疯女人》这本书似乎是围绕梅森展开的）

    简爱得知真相后感到震惊和伤心，但她毅然决然地离开这里。作者充分描写了简爱的痛苦，显得她离开的决定更加决绝。

    离开时简爱不知道去哪里，踉踉跄跄的上了一辆马车，下车时还把为数不多的钱掉车上了。她在陌生的乡村徘徊，又冷又饿，只能乞讨农家把喂猪的食物给她吃。在荒郊野外露宿了两天后，她终于找到了一户人家肯收留她。这家人有二姐妹和一个教父哥哥。

    后面的故事略显巧合，这家人竟是她父亲那边的亲戚，简爱也意外接受了父亲哥哥的遗产成了小富婆。简爱慷慨地把遗产平凡给了二姐妹和哥哥。有一天她意外得知桑菲尔德庄园失火了，梅森放的，她摔死了，而罗切斯特瞎了。她立刻赶过去找罗，二人重逢。富有的简爱，瞎了的罗，二人的地位调转，在身份上终于达到了平衡。从此罗在简爱的照顾下过上了幸福的晚年生活。story end。结尾这部分虽说有点狗血，看到二人重逢时我还是掉了几滴眼泪💧。

    Thoughts #

    简爱从小就爱看书，因为寄人篱下而对自身地位有着清晰的认知。在有机会获得教育时毫不犹豫地出手。努力学习，通过教育成为一名老师，实现了职业上的跃迁。不会因身份而尊重或诋毁别人，对每个人用平等地心交谈。爱上了贵族，而不是贵族的头衔。得知对方已婚后，勇敢地选了离开。获得了财产后没有忘记自己的本心，还是那个善良的小简爱。

    正直、勇敢、有阅读量的简爱。

    细思极恐的是如果罗的眼睛没瞎，简爱和他结婚后会不会也变成了又一个“阁楼里的疯女人”？不知道作者的本意如何，梅森到底是真疯还是被逼疯的。如果是逼疯的，再次感叹，1847年！

    摘录 #

    “我要去旅行吗？而且是跟您一起去吗，先生？”

    “你会旅居巴黎、罗马和那不勒斯，要走访佛罗伦萨、威尼斯和维也纳。那些我曾经游历过的土地，你也要畅游一遍。我的马蹄践踏过的地方，也要留下你窈窕的足迹。过去十年以来，我几近疯狂地穿越欧洲大陆，厌恶、憎恨与愤怒就是我的旅伴。如今我要带着慰藉心灵的天使，用复原又洁净的心重游故地。”

    我边听边笑。“我不是天使，”我告诉他，“在我有生之年都不会变成天使，我只会是我自己。罗彻斯特先生，您不可以期待或要求我展现任何神圣特质，因为您绝不能如愿以偿。正如同我不可能在您身上找到那些特质一样，只是我根本没有那样的期待。”

    “是吗，先生？您这么快就屈服了，看您表情多么严刺！您的眉头已经纠成一团，比我的手指头还粗。而您的额头像什么呢？像我读过的某些惊人诗句里描述的，‘层层铁青雷霆’。先生，我猜您婚后就是那副表情吧？”

    “如果你婚后就是那副表情，作为一个基督徒，也会很快放弃跟妖精或火怪相伴的念头。你到底想问什么，你这东西，说出来吧。”

    “看吧，已经不那么彬彬有礼了，但我喜欢粗鲁更胜于事，我宁可被称为东西，也不要当天使。“

    “不，你很好，最近更是进步很多，我敢说罗彻斯特先生很喜欢你，我总觉得你在他心目中很像某种宠物。曾经有好几次，我基于爱护你的心情，对他的明显偏好感到不安，很想提醒你留意一下。可是，没有真凭实据，我不愿意捕风捉影。我知道这种想法会让你很震惊，甚至觉得被冒犯，何况你一直很谨言慎行，极度谦虚又理性，我希望你懂得保护自己。昨天晚上我在屋子里上上下下找不着你，主人也不见踪影。结果，十二点左右，我看见你跟他一起进屋。你无法想象我心里多么难受。”
