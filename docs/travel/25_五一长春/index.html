    五一巨人国休息之旅

    游记
    • D1 下午飞机
    • D2 早市&姥姥家麻将
    • D3 小姨家烧烤
    • D4 姥姥家麻将&洗澡&涮羊肉
    • D5 攀岩&苏杯决赛
    • D6 回成都

    4月想去海南看姥姥请假未果，决定五一回长春看姥姥。两年没看到姥姥和外公了。他们身体还是老样子，姥姥比较硬朗，外公腿脚不好，打麻将颤颤巍巍的。

    D1 下午飞机 #

    晚上到家，老刘来接的我们，老妈在家蒸海鲜。波子吃螃蟹吃爽了。之前他总说不吃腥的，但对腥的定义很模糊，鱼吃，烤鱼片不吃，海带不吃，猪肝不吃，三文鱼吃，郡肝吃，折耳根不吃。虾爬子不怎么爱吃，螃蟹特别喜欢。

    D2 早市&姥姥家麻将 #

    每年春节回来时都想去逛早市，但不开门。这次五一回来终于赶上了。早市，是真的早。6:30我们就出门了，到了已经没什么地方停车了。这次去的应该是家附近最大的早市。在一个很大很大的棚里。内里排成3列，右边两列卖水果蔬菜，左边一列卖早饭、卤菜啥的。肉类再另一个大棚里。我们和老爸老妈分头行动，直奔卖早饭的那列。一排过去家家雾气环绕，冬天应该更热闹些。我们选了一家人比较多的店进去，点了2杯豆浆，1份豆腐脑，1根油条，1个油炸糕，一共9块！

    吃饱喝足后一到姥姥家，眼睛都快睁不开了。姥姥看我瘫在沙发上，大发慈悲地说：“困了你们就回屋睡觉吧。”五一睡觉吃饭之旅正式拉开帷幕。一回到长春身上的筋都断了，像小猫一样化成一摊水，得哪儿趴哪儿。

    睡醒后和外公打麻将。这几天打麻将打到脖子痛。

    D3 小姨家烧烤 #

    去小姨家吃席。比过年时还热闹一点，吵吵闹闹。男人们聊天喝酒吃饭，女人们干活。我是小孩，我没干活。

    得知姐姐在办离婚。他俩关系可能一直都一般。男方觉得两人互相都不想将就，没必要凑合。女方觉得和谁过日子不是凑合着过，想继续凑合。糊弄一段时间后，男方坚持离婚了。女方只能妥协。但财产分配不均。男方婚前买了个房子，婚后女方出了20万还房贷。男方以房子现在贬值了为由，说女方投资失败，只能给女方5万。女方想把钱要回来。家里人都支持离婚，但一方觉得得把钱要回来，一方觉得就当花钱让这个男的滚蛋。

    D4 姥姥家麻将&洗澡&涮羊肉 #

    又去姥姥家打麻将了，吃了排骨炖豆角，太好吃了。下午打完麻将3、4点，临时决定去洗澡。去的极乐汤，还是那么豪华。波子还搓了个澡。

    晚上特想吃涮羊肉，9点多吧老爸老妈喊出来吃。他俩还小吵一架，老妈让老爸少吃点儿，老爸说来都来了，别扫兴。实在是太好吃了！额尔顿真的比不了。肉和麻酱都很完美。不爱高钙了，口感越来越差。这次最喜欢吃大刀，肥瘦、薄厚，都很完美。

    D5 攀岩&苏杯决赛 #

    老家的攀岩馆真的太硬了。刚开始爬v1，有几条我还爬得没那么顺，爬完我想，v1变简单了呀，一看，是v0！爬真正的v1也勉强爬完了。v2居然没几条能过的！还是那么硬啊。

    抱石没什么想磕的动力，去爬难度还挺爽的。难度有15米。摸了一条10c，难度都过了，但到最后一截试了2次都掉了。春节继续回去爬难度玩。

    晚上吃了贵族螃蟹，1600元8个！居然这么贵。老爸买的时候故作潇洒，不知道心里有没有肉疼。 那个螃蟹很大，吃两个有点顶，最后的半个每个人（除了波子）都有点吃不下去。但第一个真的是人间极致享受。

    D6 回成都 #

    早上在家吃了饺子包子就开始回成都了。黑多又长胖了！开始给黑多减肥。
