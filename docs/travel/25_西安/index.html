    关于东北人爱上西安烧烤这回事

    游记

    假期戒断反应中。

    清明和波子去了西安，玩得很开心。计划是攀岩、跑步、吃很多很多碳水。实际和计划差不多，只是没怎么吃碳水，但吃了很多肉。

    行 #

    准备了一个登山包，替代行李箱和背包。正常装3天的衣服应该没啥问题，但带了攀岩鞋占了一大坨空间。背包装得满满登登，濒临超载边缘。背着体验一般，不确定是没调好还是怎么。我们的肩膀都被压红了，虽然也感受到了很大一部分力重载到腰上。买得是中号，如果是大号估计背起更重，所以这个号也算买得合适。

    这次3天的行程其实算比较紧张，因为高铁要4个小时。第一天早上7点从家出发，第三天晚上10点到家。中间睡了很多觉，所以没有很疲惫。但肌肉很酸痛！这一部分后面说。

    去时没买到普通票，坐得一等座，确实比普通座位舒服些。睡觉睡得很舒服。比起飞机、汽车，高铁是我坐起最舒服的交通工具。不会晕，可以一直看书。觉都变少了，毕竟汽车和飞机我是必睡得。座位也更宽敞。想多坐高铁去不同的城市转一转。

    西安的温度有点难把握，街上真的穿什么都有。如果中午出门短袖长裤就可以了，但晚上只穿卫衣都不够，必须得穿外套。稍一不注意就可能着凉。

    我们和波子的朋友都在这三天去西安玩。但我们没一起出发，也没住一块儿。如果时间、想吃得东西碰得上，就约到一起吃饭。能多点几个菜，还不用将就其他人的安排。不错。

    西安是围着城墙建的，城墙内的一圈，叫城里，城墙外的都是郊区（我朋友说的）。我们这次住在了看起来是1.5环靠北的地方，朋友说这是北郊。

    打车穿过城门时很奇妙，真-进城。

    食 #

    烧烤：西安的烧烤太好吃了！去之前没听说过西安烧烤，去了之后发现遍地是烧烤。当地都叫烤肉店，据说是因为最开始时只烤肉，不烤菜。现在为了适应市场发展需求，才加了一些菜进去。盐筋荣升为我最爱的烧烤品类。

    肉夹馍：波子吃了，我太撑了，没吃，他评价一般。

    羊肉泡馍：这家可能没选好，45一碗，肉好吃，但不值45。

    蒸饺（汤饺）：好吃，饺子馅加了胡椒，挺特别的。在这家店还点了八宝粥，是糊状的醪糟、枣、百合、莲子等等。红糖味的，比较甜，说不上好吃不好吃，但一大碗太扎实了，吃几口还是不错的。

    灌汤包：商场里吃得，有预制的痕迹，但味道还可以。

    乐 #

    有3个主要任务：攀岩、大明宫跑步、见我大学室友。

    攀岩选了一家离住的地方比较近的岩馆。岩馆比想象的小很多。爬得时候只有我一个女生和一个女教练，其他都是男生。新手比较多。线路很硬。只过了2条v2-v3等级。岩馆二楼有很全的健身器材，这点还不错。

    计划d2或d3哪天早上能起来就哪天早上去大明宫跑步。d2果然起不来，起来之后没多久就困了。中午吃完饭，我们又回酒店睡觉去咯。晚上去找室友吃饭，聊天聊得很开心。完全没注意到时间，就聊到很晚了。还有很多话没说完，但她第二天还要上班，就早早分开了。怀念有很多很多时间的日子。

    最后一天早上终于起来跑了，全程是7公里。我记得我之前跑过一次10公里的，所以以为7公里没什么问题。跑完波子才说，之前跑得是5公里。所以我又破记录啦！前几公里时鼻涕很多，要流出来的那种。后几公里好了一些，后面腿感觉已经有点迈不动了。本来以为能跑个7分配，最后跑了8分多。2公里后基本就是跑跑走走了。心肺能力还是太差了。真挺需要坚持一项有氧提升心肺功能的。

    西安的春天很美，虽然有些空气污染。这个季节很适合去玩。
