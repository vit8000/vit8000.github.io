    春节 #

    四人帮加波子的聚会，吃大仅火锅，吃到一半波子跑了。席间在聊春节时候大家的生活，开心的同时一半的我也想回家玩星露谷。

    小猪去了阳朔野攀，3天还是4天吃饭喝酒打车啥的花了2000块。她说以后想每年都去几次阳朔。爱上攀岩前和后的人生真的有好大的差距啊。(记得把《比山更高》带给大狗和小狗。)大狗自己去了法国十几天，太厉害了。小猪说一个人旅行时可以观察到很多两个人一起时观察不到的事情，大狗说没有啊，但她分享了很多很多故事和照片，明明观察到了很多。小狗回了云南老家，做了个快乐的猪猪。

    小猫 #

    找人上门收床，门没关，黑白吓得不知道躲到哪里去。白可跑到卧室去了，找一圈没找到黑多，我以为他跑出去了，出门找他，找了两圈都没找到。已经准备联系上门找猫了。回家时波子正给我打电话，他在家里找到黑多了。我一下子就哭了，哭了才意识到自己有多害怕。

    杂七杂八 #

    你选择看不到鸟儿还是看得到鸟儿但不能去追?

    收拾了一天屋子，把书房的床扔了，去宜家买了新的沙发床。组装完都12点了，坐在上面的时候感受到了新的一年开始了。喜欢收拾屋子了。
